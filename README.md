# Market-Memory Mamba-RAG Trader - final presentation package

This package contains presentation/report artifacts only.

## Main files

- `index.html`: polished project page. Open locally or deploy to GitHub Pages.
- `report_overleaf.tex`: Overleaf-ready LaTeX source. Upload this file together with `assets/figures/`.
- `report_overleaf.pdf`: compiled PDF report.
- `assets/figures/`: figures used by both the page and the report.
- `assets/data/component_ablation_summary.md`: ablation summary used to build the final narrative.

## Presentation claim

The clean claim is not that Mamba-RAG beats passive ETF baselines. The claim is that adding retrieval to a temporal model makes the signal auditable and improves over the no-retrieval neural baseline, while core-satellite allocation controls the aggressiveness of retrieval-only trading.

Historical backtest only; not investment advice.
